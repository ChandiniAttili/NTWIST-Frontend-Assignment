Frequently Asked Questions

What is this project about?
This is a frontend assignment for NTWIST, demonstrating HTML, CSS, and JavaScript skills.
How do I use the contact form?
Fill in your name, email, and message, then click Send Message. You'll see a confirmation and time spent.
How does the sortable list work?
Click the Sort Alphabetically button to sort the list of names without reloading the page.
Is JavaScript used on this page?
No, this FAQ accordion uses only HTML and CSS.
